Magnitie - Metal Finder Robot

This is my final project for the Intro to Embedded systems course that I took during my second year. For this project I developed a four-wheel motorized robot called Magnitie that is capable of detecting small size metal objects as it travels in a straight direction. Magnitie keeps count of the number of metals that finds as it travels, and it stops if it detects a potential obstacle in fron it. Magnitie has incorporated an ATMega 1284 micro-controller to control its entire functionality, two DC motors that enable it to move, an inductive proximity sensor to detect metals, and an ultrasonic sensor to detect obstacles. The software for this project was implemented using Atmel Studio and C programming language.

Rshell

This is a project on which I worked with a partner on the development of a C++ program that simulated the functionality of a command shell program by applying object-oriented programming principles and design patterns. We achieved the proper functionality of the program by parsing and analyzing user inputted linux commands, decomposing the commands into appropiate objects that formed a tree composite object, and executing those objects in a recursive manner using system calls. This program was progressively developed, adding extra features after unit testing the behavior of individual components by using the Google Test Framework for ++ and integration testing of the whole system using bash I/0 redirection.
